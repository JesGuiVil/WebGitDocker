COMANDOS TIPICOS DE GIT

Comando Función

git init

Inicializa un nuevo repositorio Git.

git clone url

Clona un repositorio remoto a tu máquina local.

git add archivo

Añade cambios en un archivo al área de preparación.

git commit -m "mensaje"

Guarda los cambios en el repositorio con un mensaje descriptivo.

git status

Muestra el estado actual de los archivos en el repositorio.

git log

Muestra un registro de los commits en el repositorio.

git pull

Obtiene y fusiona cambios desde un repositorio remoto.

git push

Envía los cambios locales al repositorio remoto.

git checkout nombre_rama

Cambia a una rama específica.

git merge nombre_rama

Fusiona una rama con la rama actual.

git remote -v

Muestra la URL del repositorio remoto.

git stash

Guarda temporalmente cambios no comprometidos.

git tag -a nombre_tag -m "mensaje"

Crea una etiqueta anotada en el commit actual.

git fetch

Obtiene cambios del repositorio remoto sin fusionarlos.


COMANDOS TIPICOS DE DOCKER

Comando Función

docker pull imagen

Descarga una imagen de Docker desde Docker Hub.

docker build -t nombre_imagen directorio

Construye una nueva imagen a partir de un Dockerfile.

docker run nombre_imagen

Crea y ejecuta un contenedor basado en una imagen.

docker ps

Muestra los contenedores en ejecución.

docker ps -a

Muestra todos los contenedores, incluyendo los detenidos.

docker exec -it nombre_contenedor comando

Ejecuta un comando dentro de un contenedor en ejecución.

docker stop nombre_contenedor

Detiene un contenedor en ejecución.

docker rm nombre_contenedor

Elimina un contenedor.

docker rmi nombre_imagen

Elimina una imagen de Docker.

docker images

Lista las imágenes de Docker en la máquina local.

docker-compose up

Inicia los servicios definidos en el archivo docker-compose.yml.

docker-compose down

Detiene y elimina los servicios definidos en el archivo docker-compose.yml.

docker network ls

Lista las redes de Docker.

docker volume ls

Lista los volúmenes de Docker.

docker inspect nombre_contenedor

Muestra detalles de configuración de un contenedor.

docker logs nombre_contenedor

Muestra los logs de un contenedor en ejecución.

docker cp archivo_local nombre_contenedor:directorio_contenedor

Copia archivos entre la máquina local y un contenedor.

docker-compose build

Construye o reconstruye los servicios definidos en el archivo docker-compose.yml.

docker-compose ps

Muestra el estado de los servicios en un archivo docker-compose.yml.

docker-compose exec nombre_servicio comando

Ejecuta un comando en un servicio definido en docker-compose.yml.
